Firebase — {{ currentGroup.nome }}
Ogni gruppo ha il suo Firebase dedicato e indipendente
📋 Guida rapida Firebase
1. Crea il progetto
Vai su firebase.google.com → Nuovo progetto → dai un nome (es. calcetto-giovedi)
2. Crea il Database
Menu sinistra → Realtime Database → Crea database → Modalità test → scegli la regione più vicina
3. URL Database
Copia l'URL che appare (es. https://calcetto-giovedi-rtdb.europe-west1.firebasedatabase.app) e incollalo sopra
4. Abilita autenticazione
Authentication → Inizia → Email/Password → Abilita → Aggiungi utenti con email e password
5. API Key
Impostazioni progetto (⚙️) → Generali → La tua app web → copia apiKey
6. Regole Database
Realtime Database → Regole → incolla questo:
{
  "rules": {
    ".read": true,
    ".write": "auth != null"
  }
}
💡 .read: true = tutti possono vedere i dati (consigliato per il gruppo).
.write: auth != null = solo chi ha le credenziali può modificare.
7. Aggiungere altri utenti scrittori
Authentication → Users → Aggiungi utente → inserisci email e password dell'amico che può modificare
Console Firebase → Impostazioni progetto → Generali → Web API Key
